// Checklist

Before You File an Issue

A few quick checks that help us help you faster.

1

Rescan Plugins

Quit and reopen your DAW so it re-validates AU / VST3 components. Most "missing plugin" reports clear up after a rescan.

2

Search Existing Issues

Someone may have already reported it. Browse the issue tracker — a thumbs-up on an existing issue helps us prioritize.

3

Include Details

OS and version, the Morphix build, plugin format, your host / DAW, and repro steps. Screenshots help a lot.

4

Submit

Email support or open an issue. We'll triage and respond as soon as we can.

// Troubleshooting

Common Issues

Things that come up often and how to work through them.

The plugin doesn't appear in my DAW

After installing, your host may need to rescan plugins.

  • Quit and reopen the DAW so it re-validates AU / VST3 components.
  • In Logic / GarageBand, the Audio Unit cache may need a reset (the host prompts to re-validate on launch).
  • Confirm the VST3 is in the standard folder — ~/Library/Audio/Plug-Ins/VST3 on macOS, %CommonProgramFiles%\VST3 on Windows, ~/.vst3 on Linux.

The AUv3 isn't listed on iOS

On iOS and iPadOS, an AUv3 only registers after its host app has been launched at least once.

  • Open the Morphix Standalone app once, then reopen your AUv3 host (GarageBand, AUM, Cubasis…).
  • If it still doesn't show, restart the host app so it re-scans installed Audio Units.

My factory presets are missing

The 64 factory programs ship inside the plugin and appear in the preset browser and as host programs.

  • Open the in-plugin preset browser rather than relying on the host's preset menu.
  • The bank is also written out as .vstpreset files at build time — use the plugin's Import button to load any of them manually.

Importing or exporting presets

Morphix reads and writes standard Steinberg .vstpreset files.

  • Export saves the current state as a new .vstpreset you can reload in any VST3 host.
  • Import loads a .vstpreset from disk; its class id matches the plugin so factory and user presets interchange freely.

No sound when I play

Morphix is an instrument plugin, so it needs MIDI input and an audio output route.

  • Make sure the track is armed / receiving MIDI and a note is actually being sent.
  • Check the plugin's output level and the track's gain aren't at zero.
  • Load a factory preset to rule out an all-zero patch.

Which platforms are supported?

VST3 and a Standalone app on macOS, Windows, and Linux, plus AUv3 on macOS and iOS / iPadOS. macOS builds are universal (arm64 + x86_64); Windows ships x64 and ARM64; Linux ships x86_64 and aarch64. If you hit a platform-specific issue, tell us the exact OS and version.
